First off, let's set down some definitions for sizing:

- **1 bit** = `0 OR 1`
- **1 byte** = `8 bits`
- **1 kilobyte (KB)** = `1000 bytes`
- **1 megabyte (MB)** = `1000 KB`
- **1 Gigabyte (GB)** = `1000 MB`
- **1 Terabyte (TB)** = `1000 GB`
